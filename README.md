Factoring Repository
====================


Introduction
------------

Just a varied bunch of programs that either directly factors numbers or assists with
the process.
 

Contents
--------

* WheelTF.cpp -- A simple factoring program that illustrates the wheel trial division method. 
* fermat.cpp -- Super simple implementation of Fermat's factoring algorithm.
* rho.cpp -- Super simple implementation of Pollard's Rho factoring algorithm.
